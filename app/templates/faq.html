FAQ

What is LogSense AI?

A local tool for reading system logs. Upload a log file to get structured analysis, or connect an Android device to monitor live logcat output. Runs entirely on your machine.

What file types can I upload?

.log, .txt, .xml, and .evtx files up to 64MB. The parser handles Windows Event Logs, Linux syslog, macOS unified logs, and Android logcat output. Windows .evtx binary files are parsed directly without needing to export them first.

How do I get log files from my device?

Windows: open Event Viewer, right-click a log, and save as .evtx or text. Linux: logs are in /var/log. macOS: use Console.app or the log command. The Instructions page has the full steps.

What does the threshold control do?

It adjusts how aggressively entries are classified. Strict promotes more entries to high severity. Relaxed demotes lower-confidence matches. Normal is the default and reflects the parser's built-in confidence scores.

What does the chart in the detail panel show?

Log activity across the current session. Each bar represents a group of entries colored by severity. The marked bar is the entry you selected. Hover any bar to see the triage verdict and what was happening at that point.

Do I need ADB?

Only for live Android monitoring. File upload works without ADB.

Which systems support live monitoring?

Linux, macOS, and Windows. ADB must be installed and in your PATH. On Linux: sudo apt install adb. On macOS: brew install android-platform-tools. On Windows: download Platform Tools from the Android developer site and add the folder to PATH.

My phone isn't showing up. What should I check?

Check that USB debugging is on and the phone shows as authorized when you run adb devices. On Windows you may need OEM USB drivers. On Linux you may need udev rules. Try a different cable or port if the device drops frequently.

Can I filter live logs?

Yes. The terminal has level buttons (V D I W E F) and a tag or keyword filter. Warning and Error are on by default. Turn on Info or Debug if you need more detail.

Can I save live logs?

Yes. The Save button exports all visible entries as a .txt file.

Does it work on Windows with PowerShell?

Use Command Prompt, not PowerShell. PowerShell blocks the venv activation script by default. In CMD, activate with venv\Scripts\activate and run python app.py as normal.

The live stream keeps dropping. Why?

Usually a cable or USB power issue. Try a different cable or port. Run adb devices in a terminal to check if the connection is stable.
