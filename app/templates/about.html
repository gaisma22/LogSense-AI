About LogSense AI

LogSense AI is a local log analysis tool with Android live monitoring. It runs entirely on your machine. No cloud, no accounts.

Core Features

Log Analysis

Upload a log file and get line-by-line severity classification, pattern matching, and export.

Android ADB Monitoring

Connect an Android device via USB and stream live logcat output directly into the browser.

Responsive UI

Works in dark and light mode. Built with plain HTML, CSS, and JS. No frameworks.

Architecture Overview

  • Flask backend with blueprint routing
  • Rule-based log parser with pattern matching
  • ADB device layer with lazy daemon startup
  • Server-Sent Events for live log streaming
  • In-RAM session store with automatic cleanup

Purpose

Most people never look at logs because they’re hard to read. LogSense AI shows what stands out. It does not recommend actions, deletions, or fixes.
