About LogSense AI

LogSense AI is a local log analysis tool with live Android monitoring. It runs entirely on your machine. No cloud, no accounts, no telemetry.

Features

Log Analysis

Upload .log, .txt, .xml, or .evtx files. The parser classifies entries by severity, groups them by section, and explains what each entry means.

Windows Event Logs

Reads .evtx binary files directly. Maps Event IDs to known patterns covering security, system, network, and auth events.

Android Live Monitoring

Streams live logcat output from a connected Android device via ADB. Entries are triaged in real time with hover summaries and a detail inspector.

Detail Inspector

Click any live log entry to pause the stream and inspect it. Shows triage verdict, reason, component, timestamp, and a session activity chart.

Threshold Control

Adjust how aggressively entries are classified on the results page. Strict, Normal, and Relaxed modes tune the signal-to-noise ratio for different use cases.

Fully Offline

No API keys, no cloud storage, no tracking. Analysis stays on your machine.

Architecture

  • Flask backend with blueprint routing
  • Rule-based log parser with pattern matching and confidence scoring
  • Windows Event ID lookup table covering Security, System, Sysmon, Defender, AppLocker, PowerShell, RDP, and BitLocker events
  • EVTX binary parser using the evtx package
  • ADB device layer with lazy daemon startup
  • Server-Sent Events for live log streaming
  • In-RAM session store with automatic cleanup

Purpose

Most people never look at logs because they're hard to read. LogSense AI shows what stands out. It does not recommend actions, deletions, or fixes.
